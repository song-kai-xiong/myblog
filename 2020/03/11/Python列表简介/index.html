什么是列表

通过索引读取列表中的元素，索引从0开始，-1代表最后一个元素

通过索引获取元素进行修改

添加

在列表末尾添加append()

在列表指定位置插入insert()

从列表中删除元素

被删除的元素不可以再进行访问

根据索引从列表中删除元素del

删除列表末尾的元素pop()

弹出指定位置的元素pop索引

del与pop区别：被删除的元素还能继续使用

根据元素的值进行删除remove(value)

列表排序

sort:永久排序

sorted(列表):临时排序

倒序sort(reverse=True)
